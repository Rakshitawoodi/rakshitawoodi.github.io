About

Projects

Cache and Memory Hierarchy Cache Simulator | C Programming
  • Developed four multi-level cache hierarchy models with configurable parameters, including associativity, block size, and cache size.
  • Implemented a cache hierarchy utilizing the LRU replacement policy and WBWA (Write-Back + Write-Allocate) policy.
  • Enhanced the cache subsystem by integrating stream buffers and a prefetching unit, allowing parameterization of the number and size of stream buffers.
Implementation of a Superscalar 9-Stage Out-of-Order Processor | C++
  • Designed and implemented a 9-stage out-of-order processor, encompassing stages such as fetch, decode, rename, register read, dispatch, issue, execute, write back, and retire.
  • Developed parameterized models for components like the Issue Queue Buffer and Reorder Buffer (ROB).
  • Enhanced the processor with variable width capabilities and simulated performance using two trace files across over 60 different configurations.
Hardware Implementation of Matrix Multiplication in Transformers | Verilog
  • Designed a Verilog-based hardware module for matrix multiplication in scaled dot-product attention for Transformers, utilizing SRAM for storing input, weight, and result matrices.
  • Developed a finite state machine (FSM) for efficient data flow control, ensured proper SRAM timing, and addressed read after-write (RAW) hazards.
  • Verified functionality using ModelSim and synthesized the design with Synopsys Design Compiler.
Verification of I2CMB Master with Wishbone Interface | SystemVerilog
  • Developed a layered test bench with agents, scoreboard, and coverage metrics for I2CMB Master verification, ensuring functional and code coverage using SystemVerilog.
  • Designed and integrated an I2C interface package, implementing transfer detection, data handling, and monitoring within a structured verification flow.
  • Executed 64 read/write transactions, leveraging assertions, self-checking mechanisms, and transcript-based validation for protocol compliance.
